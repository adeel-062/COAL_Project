# COAL_Project
A scientific calculator code written in assembly language using the irvine library.
Included operations are basic arithmetic (addition, subtraction, multiplication, division), scientific calculations (exponentials, logaraithmics, trignometric funtions, quadratic equations), vector addition and scalar multiplication with added axiom checks to determine whether a set forms a vector space or not. 
Proper error handling (where applicable) has also been implemented.
